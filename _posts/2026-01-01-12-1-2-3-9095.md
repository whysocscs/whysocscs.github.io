<p data-ke-size="size16">DP 문제를 풀어보기 위해서 뭔가 간단해 보이는 문제를 골랐다.</p>
<p data-ke-size="size16"> </p>
<p data-ke-size="size16">DP 문제를 풀기 위해서는 점화식 설계를 잘 해야하므로 노트에 적어가면서 풀었다.</p>
<p data-ke-size="size16">1~5까지 쭉 숫자들을 나열해 보니까</p>
<p data-ke-size="size16">숫자 5는</p>
<p data-ke-size="size16">숫자 4가 가진 경우의 수에 +1을 하면 5가 나오고</p>
<p data-ke-size="size16">숫자 3이 가진 경우의 수에 +2를 하면 5가 나오고</p>
<p data-ke-size="size16">숫자 2가 가진 경우의 수에 +3을 하면 5가 나온다.</p>
<p data-ke-size="size16"> </p>
<p data-ke-size="size16">즉 F(n) = F(n-1)+F(n-2)+F(n-3)에 대한 점화식이 나온다.</p>
<p data-ke-size="size16"> </p>
<pre class="cpp" data-ke-language="cpp" data-ke-type="codeblock" id="code_1767278018692"><code>#define _CRT_SECURE_NO_WARNINGS
#include&lt;stdio.h&gt;

int easy[11];


int main() {
	easy[0] = 0; easy[1] = 1; easy[2] = 2; easy[3] = 4; easy[4] = 7;
	for (int i = 5; i &lt;= 10; i++) {
		easy[i] = easy[i - 2] + easy[i - 1] + easy[i - 3];
	}
	
	int cnt=0, n=0;
	scanf("%d",&amp;cnt);
	for(int i=0;i&lt;cnt;i++) {
		scanf("%d", &amp;n);
		printf("%d\n", easy[n]);
	}
	
}</code></pre>
